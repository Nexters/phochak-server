== 게시글 APIs

=== 게시글 불러오기
조회 기준(포착순, 최신순, 조회순)에 따라 원하는 size만큼 게시글을 불러옵니다.

==== 발생 가능 서비스 예외 코드
"P412", "최신순이 아닌 경우 정렬기준의 값은 필수입니다"

"P413", "지원하지 않는 정렬기준입니다"

"P414", "정렬 기준이 존재하지 않습니다"
